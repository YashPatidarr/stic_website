WEBDROID

Intro Duction to Webdroid.

Webdroid is the club of STIC devoted entirely for computer engineering and its sub fields. Here, we deal with the study of the theory, experimentation, and engineering that forms the basis for the design and use of computers. In webdroid we emphasis on implementation of theoretical knowledge. Through webdroid we provide its members a platform to work on real time projects, communication, workplaces, tutorial classes and workshops.

DIVISIONS


Webcoders

The group is about web application development like website development, developing, online games, developing web apps for android.

Theme : Clean web user interface. 

Projects :

  • Developing an enhanced version of Mario (game) JavaScript based.
  • Developing an updated website for the university.

Programmers Arena

The group is about local and international level coding competitions. We will help the member’s collaborate and form teams for different coding competitions.

Theme : ICPC 


Developer Zone
